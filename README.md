# DarkPiumosso

DarkPiumosso - color scheme for Pycharm

## Installation

```
$ git clone git://github.com/piumosso/DarkPiumosso.git
For MacOS:
$ cp DarkPiumosso/*.xml ~/Library/Preferences/PyCharm20/colors
For Linux:
$ cp DarkPiumosso/*.xml ~/.PyCharm20/config/colors
```
